# Social Links Profile - Frontend Mentor

Welcome to the Social Links Profile repository! This project is a personal profile card designed to showcase social media links. It is a clean, fully responsive component built using semantic HTML and modern CSS.

## Features

- **Responsive Design:** The layout adapts perfectly to various screen sizes (Mobile, Tablet, and Desktop).
- **Hover States:** Interactive elements feature hover effects for a better user experience.
- **Semantic HTML:** Built with accessibility and SEO in mind using proper tags.
- **Custom CSS Properties:** Used CSS variables for easy color management and consistency.

## Technologies Used

- HTML5
- CSS3 (Flexbox & CSS Variables)
